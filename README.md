# 🚀 AI Data Extraction & Automation System

> Transform messy PDFs, Excel files, and CSV data into clean, structured outputs using AI 🤖

---

## 📌 Overview

This project is an **AI-powered data processing system** that extracts, cleans, validates, and transforms semi-structured data into structured formats.

It combines **Python pipelines + LLM intelligence + Streamlit UI** to automate real-world data workflows.

---

## 🎯 Problem Statement

Businesses deal with:
- ❌ Unstructured PDFs  
- ❌ Messy Excel sheets  
- ❌ Manual data entry  

👉 This leads to errors, inefficiency, and wasted time.

---

## 💡 Solution

✅ Automated data extraction  
✅ Intelligent processing using LLMs  
✅ Rule-based validation  
✅ Clean structured output (CSV/JSON)  
✅ Simple UI for easy interaction  

---

## ⚙️ Features

✨ Extract data from:
- PDFs 📄  
- Excel 📊  
- CSV files 📁  

✨ Data Processing:
- Cleaning & transformation  
- Validation engine  
- Error handling  

✨ AI Integration:
- LLM-based extraction (GPT)  
- Prompt engineering  

✨ UI:
- Streamlit-based interface  
- Upload → Process → Download  

---

## 🧠 Tech Stack

- 🐍 Python  
- 📊 Pandas  
- ⚡ FastAPI  
- 🎨 Streamlit  
- 🤖 OpenAI API  

---

## 🖥️ Demo Flow

1. Upload file (PDF/CSV/Excel)  
2. Click **Process**  
3. System extracts & cleans data  
4. Download structured output  

---

## ▶️ How to Run

### 1️⃣ Install dependencies
```bash
pip install -r requirements.txt
````

### 2️⃣ Run backend (FastAPI)

```bash
python3 -m uvicorn app:app --reload --port 8001
```

### 3️⃣ Run UI (Streamlit)

```bash
streamlit run ui.py
```
